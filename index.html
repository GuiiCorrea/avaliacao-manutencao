Respostas da Avaliação de Manutenção de hardware

1. O que é hardware?
B) Parte física do computador
2. A função da placa-mãe é:
C) Conectar todos os componentes
3. O processador é responsável por:
B) Executar instruções
4. Sobre memórias, marque a correta:
C) RAM é volátil
5. Dispositivos de entrada são:
B) Teclado e mouse
6. A BIOS é responsável por:
B) Inicializar o hardware
7. Qual componente fornece energia ao computador?
B) Fonte
8. Sobre manutenção preventiva:
B) Evita problemas futuros
9. A formatação de um computador serve para:
C) Instalar ou reinstalar o sistema
10. Um exemplo de dispositivo de saída é:
C) Monitor
11. O sistema operacional tem como função:
A) Controlar o hardware e software
12. Sobre redes de computadores:
B) Permitem comunicação entre computadores
13. Explique a diferença entre hardware e software.
Hardware são as partes físicas que você pode tocar. Software são os programas que fazem o computador funcionar.
14. Descreva a função da placa-mãe no funcionamento do computador.
A placa-mãe liga todas as peças do computador e permite que elas trabalhem juntas.
15. Explique o que acontece quando um computador é ligado (processo inicial).
Quando liga, o computador verifica se está tudo funcionando e depois inicia o sistema operacional.
16. Diferencie memória RAM e armazenamento (HD/SSD).
A RAM é temporária e usada enquanto o computador está ligado. O HD ou SSD guarda os arquivos de forma permanente.
17. O que é manutenção corretiva? Dê um exemplo.
É quando você conserta algo que já deu problema, como trocar uma peça que queimou.
18. Explique a importância da limpeza interna do computador.
A limpeza evita poeira, superaquecimento e ajuda o computador a durar mais.
19. Descreva os passos básicos para montar um computador.
Montar as peças principais, conectar os cabos e ligar tudo corretamente.
20. Explique por que é importante instalar corretamente um sistema operacional.
Porque assim o computador funciona bem, sem erros ou travamentos.
